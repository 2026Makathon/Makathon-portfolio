## .exe 실행 방법

Beat Maple과 PacMan은 Java로 만들어졌기 때문에 **Java 17 이상**이 PC에 설치돼 있어야 실행됩니다.

1. 아래 링크에서 본인 OS에 맞는 설치 파일을 받습니다.
   - Windows: <https://adoptium.net/temurin/releases/?version=17&os=windows>
   - macOS: <https://adoptium.net/temurin/releases/?version=17&os=mac>
2. 설치 후 .exe 파일을 더블클릭합니다.
3. Windows에서 "Windows의 PC 보호" 경고가 뜨면 **추가 정보 → 실행** 순서로 눌러주세요.
   (배포 인증서가 없는 개인 빌드라 발생하는 정상 메시지입니다.)
